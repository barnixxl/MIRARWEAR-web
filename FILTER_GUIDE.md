# Руководство по системе фильтров одежды

## Обзор системы

Система фильтров позволяет пользователям искать товары по:
- **Категории одежды**: шапка, кофта, майка, штаны, носки, нижнее белье, куртка, шорты
- **Размеру**: S, M, L, XL, единый размер
- **Цене**: до указанной суммы

## Как использовать фильтры

### Для пользователей:
1. На главной странице нажмите кнопку "Фильтры"
2. Выберите нужные категории одежды (можно несколько)
3. Выберите размеры (можно несколько)
4. Установите максимальную цену с помощью слайдера
5. Нажмите "Показать товары"
6. Система покажет только товары, соответствующие выбранным критериям

### Для администраторов:
1. Войдите в админ-панель (admin.html)
2. Перейдите в раздел "Товары"
3. Нажмите "Добавить товар"
4. Заполните форму:
   - Название товара
   - Описание
   - **Категория** (обязательно выбрать из списка)
   - Цена
   - Количество
   - Размеры (включить/выключить нужные)
   - Загрузить изображения
5. Нажмите "Сохранить"

## Категории товаров

- **hat** - Шапка
- **sweater** - Кофта  
- **t-shirt** - Майка
- **pants** - Штаны
- **socks** - Носки
- **underwear** - Нижнее белье
- **jacket** - Куртка
- **shorts** - Шорты

## Примеры товаров

Система автоматически создает примеры товаров при первом запуске:
- Классическая шапка (150 Br)
- Удобная кофта (800 Br)
- Спортивная майка (300 Br)
- Джинсы классические (1200 Br)
- Теплые носки (100 Br)
- Семейные трусы (200 Br)
- Демисезонная куртка (1500 Br)
- Летние шорты (400 Br)

## Технические детали

- Все данные хранятся в localStorage браузера
- Фильтрация происходит на клиентской стороне
- Поддерживается множественный выбор категорий и размеров
- Система автоматически инициализирует примеры товаров при первом запуске
